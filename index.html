jQuery Sapling

jQuery Sapling provides you a base CSS & JavaScript code in order to help you to create your own customized tree/accordion lists very quickly.

It do not offer any complex functionality, on the contrary, it is simple (Just 1.6 KB) and its visual richness is up to how you customized its CSS. If you've looked for more competent plugins, you may check jQuery Treeview (8,1 KB), jsTree (180 KB) and of course jQuery UI.

Usage

Requires jQuery 1.4.2 or later.

$("#myUnorderedList").sapling();
